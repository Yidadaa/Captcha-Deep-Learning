## Recognize Captcha Images
> Python 2.x + PIL + tensorflow

### Usage
```bash
# 0. Install dependencies
pip install -r requirements.txt

# 1. Download dataset
python download.py

# 2. Start training your model
python captcha.py

# 3. Evaluate your model
python test.py
```
